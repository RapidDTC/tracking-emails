== Description ==

= Notify customers =
It's easy to notify customers that their package is ready to be sent.

= Save tracking numbers =
Save tracking numbers in your cockpit to easily manage them

= Links =
Quick links to pages, with tracking.

The plugin may contain links not only to official couriers websites.
If we could not find the right url, the 'Track the package' button will refer to the 17track.net

== Installation ==

= Minimum Requirements =

* PHP version 5.4.0 or greater (PHP 7.2 or greater is recommended)
* MySQL version 5.0 or greater (MySQL 5.6 or greater is recommended)
* WordPress version 4.5.0 or greater
* WooCommerce version 3.4.0 or greater

= Automatic installation =

Automatic installation is the easiest option as WordPress handles the file transfers itself and you don’t need to leave your web browser. To do an automatic install of Email tracking notification for WooCommerce, log in to your WordPress dashboard, navigate to the Plugins menu and click Add New.

In the search field type Email tracking notification for Email tracking notification for WooCommerce and click Search Plugins. Once you’ve found our tracking plugin you can view details about it such as the point release, rating and description. Most importantly of course, you can install it by simply clicking “Install Now”.

= Manual installation =

The manual installation method involves downloading our tracking plugin and uploading it to your webserver via your favourite FTP application. The WordPress codex contains [instructions on how to do this here](https://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation).

= Updating =

Automatic updates should work like a charm; as always though, ensure you backup your site just in case.

== Frequently asked questions ==

= What features does this plugin have? =

By using this plugin you can send tracking information to your clients.

= How do I go to settings? =

The tracking options are added in the right section of each order

= Does the plugin support many languages? =

This plugin contains a full translation into Polish and English.

The plugin has integrated protection against outdated versions of WordPress and PHP.
