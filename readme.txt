Send e-mail notifications about the delivery status to your customers. It's easy!

== Description ==

= Notify customers =
Notify customers that their package is ready to be sent.

= Save tracking numbers =
Save tracking numbers in your cockpit to easily manage them

= Customer information =
Customer sees his package number and tracking link in his order details

= Links =
Quick links to pages, with tracking.

The plugin may contain links not only to official couriers websites.
If we could not find the right url, the 'Track the package' button will refer to the 17track.net

= Carriers =
The list of carriers includes:
USPS, FedEx, UPS, EMS, DHL, TNT, India Post, DPD, GLS, PostNL, Schenker, Royal Mail, Blue Dart, Japan Post, InPost, Poczta Polska and Envelo

== Installation ==

= Minimum Requirements =

* PHP version 5.4.0 or greater (PHP 7.2 or greater is recommended)
* MySQL version 5.0 or greater (MySQL 5.6 or greater is recommended)
* WordPress version 4.5.0 or greater
* WooCommerce version 3.4.0 or greater

= Automatic installation =

Automatic installation is the easiest option as WordPress handles the file transfers itself and you don’t need to leave your web browser. To do an automatic install of our plugin, log in to your WordPress dashboard, navigate to the Plugins menu and click Add New.

In the search field type 'Tracking: Emails and Notifications for WooCommerce' and click Search Plugins. Once you’ve found our plugin you can view details about it such as the point release, rating and description. Most importantly of course, you can install it by simply clicking “Install Now”.

= Manual installation =

The manual installation method involves downloading our plugin and uploading it to your webserver via your favourite FTP application. The WordPress codex contains [instructions on how to do this here](https://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation).

= Updating =

Automatic updates should work like a charm; as always though, ensure you backup your site just in case.

== Frequently asked questions ==

= What features does this plugin have? =

By using this plugin you can send tracking information to your clients.
You can also save tracking numbers in your order.
The customer can see the tracking number on the preview page of his order

= How do I go to settings? =

The tracking options are added in the right section of each order

= Does the plugin support many languages? =

This plugin contains a full translation into Polish and English.

= Does the plug have any security features? =

The plugin has integrated protection against outdated versions of WordPress, WooCommerce and PHP.
It also detects if the WooCommerce plugin has been disabled

= Do the creators of the plugin collect any information about me? =

No, the plugin and its components work only on your site..
